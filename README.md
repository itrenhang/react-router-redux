# react-router-redux
platform
NOBOOK新版后台管理项目

**在运行前请确保你的电脑上安装了node**

[node安装方法](http://www.runoob.com/nodejs/nodejs-tutorial.html)

一般 mac 自带node

在终端运行命令

```
node -v
```

可以查看node版本

-----------------------------

## 项目运行

* 在终端运行命令

```
npm install
```

* 项目运行

```
npm run dev
```
* 打开浏览器，输入`localhost:8080`

* 项目打包

```
npm start
```
